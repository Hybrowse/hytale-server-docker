## Scope

This document describes how the **Hybrowse** name, logos, and branding may be used in connection with this repository and related Hybrowse software.

This repository is an **unofficial** community project and is not affiliated with or endorsed by Hypixel Studios Canada Inc.

- This repository is not an official Hytale product.

## Hybrowse trademarks

"Hybrowse" and related Hybrowse logos/branding are trademarks of Arens & Gerstner GbR.

### Allowed (without permission)

You may:

- Use the name "Hybrowse" to **truthfully refer** to Hybrowse as the author/maintainer of this project.
- Link to the official repositories, website, and documentation.
- State compatibility or usage in a factual way (for example: "works with the Hytale Server Docker Image by Hybrowse").

### Not allowed (without permission)

You may not:

- Use Hybrowse logos/branding in a way that suggests **endorsement or affiliation**.
- Name a fork or derivative in a way that is confusingly similar to "Hybrowse" or implies it is the official Hybrowse release.
- Use Hybrowse branding in marketing materials for a commercial hosting/managed service without written permission.

## Official builds

If you distribute builds, images, or hosted services based on this project, you must not represent them as "official Hybrowse" unless you have written permission from Hybrowse / Arens & Gerstner GbR.
